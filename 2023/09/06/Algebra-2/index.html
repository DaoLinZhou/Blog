Ring and Fields

环(Ring) 是一类包含两种运算(加法和乘法)的代数系统

域(Field) 是一种具有加法跟乘法的集合，且其加法跟乘法运算就如同普通的有理数还有实数


Natural Number

$\N$ 是有顺序的, 例如 2 < 3

$N$ 是 well-ordered: 对于任何非空子集 $S\subseteq \N$, S 总有一个最小值

例如:

has smallest element 1


Theorem (Strong induction):

让 $X \subseteq\N$, 如果满足

  1. $0 \in X$
  2. 如果 $n \in \N$, 且 $m<n \implies m\in X$, 那么 $n\in X$

那么 $X= \N$


应用例子:


证明 Strong Induction

假设 $X\ne \N$, 那么 就让 $Y=\N\backslash X$

by well-ordering of $\N$, $Y$ 有一个最小值 $n$

由于 $0\in X$, $n>0$

考虑 $m\in \N$ 且 $m<n$ 那么 $m\in X$, 根据 $X$ 的第二个性质 $n\in X$

矛盾, 因此 $X = \N$

乘法

我们可以用 + 加法来定义乘法

$a\cdot b$ 的定义是 $0 \cdot b=0, \forall b\in\N$

同时 $\forall a, b \in \N$

我们可以用induction 来证明这个定义 uniquely determines $a\cdot b, \forall a, b \in \N$


证明: $\forall a, b \in \N$

  1. $a\cdot b = b\cdot a$ (commutativity)
  2. $(a\cdot b) \cdot c = a\cdot (b\cdot c)$ (associativity)


证明

lemma: $a\cdot 0 = 0 , \forall a \in \N$

Induction on $a$

假设 $a > 0, a = a’+1$

$a\cdot 0 = (a’+1)\cdot 0 = a’\cdot 0 + 1\cdot 0 = 0+0=0$


证明1: $a\cdot b = b\cdot a$ (induct on a)

假设 $a,b>0 , \qquad a=a’+1,\qquad b=b’+1$
