TensorFlow 2 学习笔记

写一个简单的神经网络

from sklearn.preprocessing import StandardScaler

from tensorflow import keras


if __name__ == '__main__':
    # 获取数据集
    fashion_mnist = keras.datasets.fashion_mnist
    (x_train_all, y_train_all), (x_test, y_test) = fashion_mnist.load_data()
    x_valid, x_train = x_train_all[:5000], x_train_all[5000:]
    y_valid, y_train = y_train_all[:5000], y_train_all[5000:]

    # 预处理, 方差归一化
    stand_scaler = StandardScaler()
    stand_scaler.fit(x_train.reshape(-1, 1))
    x_train_scaled = stand_scaler.transform(x_train.reshape(-1, 1)).reshape(-1, 28, 28)
    x_valid_scaled = stand_scaler.transform(x_valid.reshape(-1, 1)).reshape(-1, 28, 28)
    x_test_scaled = stand_scaler.transform(x_test.reshape(-1, 1)).reshape(-1, 28, 28)

    # 建立模型
    model = keras.models.Sequential()
    model.add(keras.layers.Flatten(input_shape=[28, 28]))    # 把2维图片铺平为一维向量
    model.add(keras.layers.Dense(400, activation="relu"))
    model.add(keras.layers.Dense(100, activation="relu"))
    model.add(keras.layers.Dense(10, activation="softmax"))

    # 设置使用的损失函数, y 原本是index值, 需要用 onehot变成向量才行
    # 所以用 sparse_categorical_crossentropy
    # 而如果 y 本身就是 onehot 可以直接用 categorical_crossentropy, 
    model.compile(loss="sparse_categorical_crossentropy",
                  optimizer="adam",   # 模型求解方法
                  metrics=["accuracy"])
    history = model.fit(x_train_scaled, y_train, epochs=10, validation_data=(x_valid_scaled, y_valid))

    # 信息
    print(model.summary())
    print(history.history)

批归一化

归一化是指在训练之前, 对数据进行一次归一化.

而批归一化指的是在神经网络的每一层都进行一次归一化.

# 批归一化放在激活函数之后
for _ in range(20):
    model.add(keras.layers.Dense(100, activation="relu"))
    model.add(keras.layers.BatchNormalization())    # 归一化
# 批归一化放在激活函数之前
for _ in range(20):
    model.add(keras.layers.Dense(100))
    model.add(keras.layers.BatchNormalization())
    model.add(keras.layers.Activation("relu"))

dropout

随机的让某些单元失效(每一次训练弃用的单元都是随机的). 这样节点就无法组合地去学习, 每个节点必须独立学习数据的规律. 这样就可以防止过拟合

model.add(keras.layers.AlphaDropout(rate=0.5))
model.add(keras.layers.Dropout(rate=0.5))

dropout在tf中也是一层, 是对前一层的节点进行dropout


AlphaDropout 比 Dropout更优秀, 它可以保证均值方差不变, 因此归一化的性质也不变.


Wide & Deep 模型

这个模型可以用于分类问题也可以用于回归问题

这个模型把数据分为稀疏特征和密集特征两种, 基于两种特征构建模型

  • 稀疏特征
    • 离散值特征
    • 可以用one-hot表示
    • 例如: 专业 = {计算机, 人文, 其他}. 人文 = $[0, 1, 0]$
    • 例如: 词表={人工智能, 你, 它}, 它 = $[0, 0, 1]$
    • 叉乘: {(计算机, 人工智能), (计算机, 你) …}
  • 密集特征
    • 向量表达, 例如 专业 = {计算机, 人文, 其他}. 人文 = [0.3, 0.2], 此时就可以用向量之间的距离来表示两个词之间的距离
    • word2vec 工具就可以把单词转换为向量
      • 男-女 = 国王-王后


此时就不能使用 Sequential 了, 因为此时一层有两种模式

input = keras.layers.Input(shape=x_train.shape[1:])

model.compile(loss="sparse_categorical_crossentropy",
              optimizer="adam",
              metrics=["accuracy"])

另一种方法是重写子类

# 函数式 API
class WideDeepModel(keras.models.Model):
    def __init__(self):
        super(WideDeepModel, self).__init__()
        """定义模型层次"""
        self.hidden1_layer = keras.layers.Dense(30, activation="relu")
        self.hidden2_layer = keras.layers.Dense(30, activation="relu")
        self.output_layer = keras.layers.Dense(1)

    def call(self, input):
        """完成模型的正向运算"""
        hidden1 = self.hidden1_layer(input)
        hidden2 = self.hidden2_layer(hidden1)
        concat = keras.layers.concatenate([input, hidden2])
        output = self.output_layer(concat)
        return output

model = WideDeepModel()
model.build(input_shape=(None, 8))    # None 是数据的个数, 8 是数据有多少特征

model.compile(loss="mean_squared_error",
              optimizer="adam",
              metrics=["accuracy"])

history = model.fit(x_train_scaled, y_train, epochs=50, validation_data=(x_valid_scaled, y_valid))

随机搜索

如果数据是连续的, 那么有可能网格搜索会跳过某个最优参数. 而使用随机搜索就可以使得最优解有概率被访问到.


遗传算法

遗传算法是模拟自然界的遗传.

它的思想是这样:

  1. 初始化候选参数集合 -> 训练 -> 得到模型指标(把这个指标当作生存概率)
  2. 基于指标(生存概率)进行随机选择
    • 把参数进行交叉. (选择一个参数集合中的一些参数, 再选择另外一个参数集合中的一些参数组合成一个新的参数集合).
    • 变异: 对某几个参数进行微小的调整 (DNA除了来自父母, 还会有变异的一部分)
    • 产生下一代集合
  3. 回到 (1) 进行下一轮

def build_model(hidden_layers=1,
                layer_size=30,
                learning_rate=3e-3):
    model = keras.models.Sequential()
    model.add(keras.layers.Dense(layer_size, activation="relu",
                                 input_shape=x_train.shape[1:]))
    for _ in range(hidden_layers - 1):
        model.add(keras.layers.Dense(layer_size, activation="relu"))
    model.add(keras.layers.Dense(1))
    optimizer = keras.optimizers.SGD(learning_rate)

卷积神经网络

卷积神经网络的核心就是卷积. 通常卷积神经网络和卷积操作用于处理图像.

可以把卷积操作看作是对图像的预处理, 再通过神经网络分析卷积后的结果.

不过卷积核再网络中是可变的, 也就相当于由卷积神经网络自己判断要怎么对图像预处理.


卷积神经网络的问题

参数过多

如果图像是(1000, 1000)

假设下一层神经元的个数是$10^6$

那么全连接参数就是输入乘输出$1000100010^6=10^{12}$

这么大的数据量可能会导致计算资源不足, 且容易过拟合

解决方法之一就是使用局部卷积. 一个单元只和图像中的一部分进行连接.


如果局部连接范围是(10, 10)

那么全连接参数为$101010^6=10^8$

虽然减少很多参数量, 但是参数量依然很大


另一个方法就是参数共享

不同的局部连接的参数是一样的

例如图中红色单元与(10, 10)的区域作连接, 有100个参数.

蓝色单元同样与(10, 10) 的区域作连接, 有100个参数. 但是我们让它与红色单元是同一组100个参数

此时全连接参数为$10*10 = 100$



为什么参数共享起作用?

这是因为图像的区域性, 图像特征和位置无关

例如一个脸无论是在图片的左上方还是在图片的右下方, 它都应该是一张脸.



卷积神经网络实现

无论原图有多少channel, 在使用同样channel数量的卷积核卷积后channel都变为1. 因此我们多设置几个卷积核, filter=32就说明有32个卷积核, 卷积后的结果就有32个channel.

    y_valid, y_train = y_train_all[:5000], y_train_all[5000:]
    scaler = StandardScaler()
    x_train_scaled = scaler.fit_transform(
        x_train.astype(np.float32).reshape(-1, 1)).reshape(-1, 28, 28, 1)
    x_valid_scaled = scaler.transform(
        x_valid.astype(np.float32).reshape(-1, 1)).reshape(-1, 28, 28, 1)
    x_test_scaled = scaler.transform(
        x_test.astype(np.float32).reshape(-1, 1)).reshape(-1, 28, 28, 1)

    # 模型, 卷积神经网络
    # 两层卷积, 一层池化
    model = keras.models.Sequential()
    model.add(keras.layers.Conv2D(filters=32, kernel_size=3,
                                  padding="same",
                                  activation="relu",
                                  input_shape=(28, 28, 1)))
    model.add(keras.layers.Conv2D(filters=32, kernel_size=3,
                                  padding="same",
                                  activation="relu"))

    model.add(keras.layers.MaxPool2D(pool_size=2))
    model.add(keras.layers.Conv2D(filters=64, kernel_size=3,
                                  padding="same",
                                  activation="relu"))
    model.add(keras.layers.Conv2D(filters=64, kernel_size=3,
                                  padding="same",
                                  activation="relu"))
    model.add(keras.layers.MaxPool2D(pool_size=2))
    model.add(keras.layers.Conv2D(filters=128, kernel_size=3,
                                  padding="same",
                                  activation="relu"))
    model.add(keras.layers.Conv2D(filters=128, kernel_size=3,
                                  padding="same",
                                  activation="relu"))
    model.add(keras.layers.MaxPool2D(pool_size=2))
    model.add(keras.layers.Flatten())
    model.add(keras.layers.Dense(128, activation="relu"))
    model.add(keras.layers.Dense(10, activation="softmax"))

    model.compile(loss="sparse_categorical_crossentropy",
                  optimizer="sgd",
                  metrics=["accuracy"])
    callbacks = [keras.callbacks.EarlyStopping(patience=5, min_delta=1e-3)]
    history = model.fit(x_train_scaled, y_train, epochs=10,
                        validation_data=(x_valid_scaled, y_valid),
                        callbacks=callbacks)

因为池化会导致长宽缩小为原来的1/2, 有一些信息损失了

为了缓解这种损失, 我们把filter翻倍

为什么要这样作分支?

这样可以有不同尺寸视野域

视野域:

  • 不同的视野域提取的图像特征尺度不同

例如最顶上的那个方块的视野域是(5, 5)的方阵, 而中间层的每一个方块的视野域是(3, 3)的区域

视野域可以叠加, 两层3x3的视野域就是5x5


而在这个结构中, 每个分支视野域是不一样的.

从左到右视野域分别是(1, 1), (3, 3), (5, 5)(average pool会是图像宽高变为原来的1/2), (5, 5)



深度可分离卷积也是一种分支的网络结构, 只不过它的分支是建立在不同的网络通道之上.

对于输入, 可以经过一个1x1的卷积得到一个多通道的输出. 我们把这个输出分为3份分别输入给3个不同的分支, 再把不同结果进行合并

如果分到极值: 把每个通道都分成一份


而且深度可分离卷积的计算速度相对普通卷积快


普通卷积的计算量 $D_KD_KMND_F*D_F$

$D_K$ 核的宽高(通常为正方形所以是$D_K^2$)

$D_F$ 是图的宽高

$M$是图像的channel数(输入的channel数)

$N$是输出的channel数

对单个像素进行卷积就是$D_KD_KM$, 对于一个卷积核来说, 就是 $D_KD_KMD_FD_F$

如果有N个filter, 那么就是 $D_KD_KMND_F*D_F$


深度可分离卷积的计算量 $D_KD_KMD_FD_F$


1x1卷积的计算了则是 $MND_F*D_F$ 因为 $D_K=1$


优化比例就是:
$$
\frac {D_KD_KMD_FD_F+MND_FD_F} {D_KD_KMND_FD_F}
$$

$$
=\frac {1}{N}+\frac {1}{D_K^2}
$$

这个值就是深度可分离卷积的计算量与普通卷积计算量的比值.

这个值越小, 说明优化越大



要在tensorflow中使用深度可分离卷积需要使用SeparableConv2D 而不是Conv2D

model = keras.models.Sequential()
model.add(keras.layers.SeparableConv2D(filters=32, kernel_size=3,
                                       padding="same",
                                       activation="relu",
                                       input_shape=(28, 28, 1)))
model.add(keras.layers.SeparableConv2D(filters=32, kernel_size=3,
                                       padding="same",
                                       activation="relu"))

model.add(keras.layers.MaxPool2D(pool_size=2))
