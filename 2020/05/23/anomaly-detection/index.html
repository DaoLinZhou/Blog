异常检测 Anomaly Detection

异常检测就是发现离群点, 有时也被称为偏差检测

就是寻找出偏离集群的点


既可以单独作为一个分类方法使用, 也可以作为其他算法的辅助使用(过滤outlier)


异常检测有非常多的方法, 这里介绍一个比较常用的基于正态分布模型的方法

从一维数据看起

假设有一组一维数据
$$
{ x^{(1)},x^{(2)},\cdots,x^{(m)} }
$$
映射到数据轴上就是

左右两边的数据密度比较稀疏, 中间的数据密度比较集中.

通常服从正态分布

也就是用正态分布作为概率密度函数


概率密度函数是一个描述随机变量在某个确定的取值点附近的可能性函数

p(x1, x2)就是指的x1 到 x2 部分的阴影面积.
$$
P(x_1, x_2)=\int_{x_1}^{x_2} p(x) dx
$$
如果(x1, x2) = (-inf, inf), 那么整个阴影面积就是1


正态分布的p(x)为
$$
p(x) = \frac 1 {\sqrt{2\pi}\sigma}e^{-\frac {(x-\mu)^2}{2\sigma^2}}
$$

将数据点转化为标准正态分布之后, 就可以设立一个阈值ε

将数据点代入式子中求p(x), 如果p(x) < ε, 就说明这个点是异常点.


当数据高于一维

通常我们处理的数据都是多维度的, 此时应该怎么做?

此时就对每一个维度计算它的概率密度
$$
p(x^{(i)})= \frac 1 {\sqrt{2\pi}\sigma}e^{-\frac {(x^{(i)}-\mu)^2}{2\sigma^2}}
$$
μ和σ是当前维度 i 的均值和标准差


整体的概率密度就是各个维度的概率密度的乘积
$$
p(x)=\prod_{j=1}^n p(x_j;\mu_j, \sigma_j)
$$

$$
\prod 代表乘积,\ \prod_{i=1}^n a_i = a_1a_2\cdots a_n
$$


代码实现

import numpy as np
import matplotlib.pyplot as plt
from sklearn import datasets
from sklearn.preprocessing import StandardScaler

# 计算X对应正态分布的y值
def normal_distribution(X):
    return np.exp(-X**2 / 2) / np.sqrt(2 * np.pi)

# 计算概率密度函数
def p(X):
    p = 1
    for i in range(X.shape[1]):
        p *= normal_distribution(X[:, i])
    return p

# epsilon 默认取3sigma法则对应的y值
def anomaly_detection(X, epsilon=0.0044):
    # 归一化, 使得数据服从z分布
    new_X = StandardScaler().fit(X).transform(X)
    return p(new_X) <= epsilon


if __name__ == '__main__':
    centers = [[3, 3], [-4, -5], [-1, 2.5], [2, -2]]
    X, _ = datasets.make_blobs(n_samples=1000, centers=centers, cluster_std=0.8)
    outlier = np.array([[6, -6], [6, -5], [2, -6], [-4, -6], [4, -2],
                        [-7, 4], [-6.5, 6]])
    X = np.vstack([X, outlier])

    anomaly_data = X[anomaly_detection(X, 0.007), :]

    plt.scatter(X[:, 0], X[:, 1])
    plt.scatter(anomaly_data[:, 0], anomaly_data[:, 1], color="red")
    plt.show()

然而设置阈值也是比较麻烦的. 与其指定阈值, 不如指定百分比, 通过百分比找阈值.

例如我们要找出10%个outlier, 就先求出所有x的概率密度函数y, y呈正态分布

因此找到最小的10%个y就找到outlier, 以最小的10%个y中最大的一个y作为分界点. 以此来predict


因此模仿scikit-learn中的EllipticEnvelope类

同时更改predict, 使得 1为正常值, -1为异常值

class MyEllipticEnvelope:

    # 归一化
    def standard(self, X):
        means = np.mean(X, axis=0)
        std = np.std(X, axis=0)
        return (X-means)/std

    #  contamination 指定筛选出多少outlier, 0.1就是10%的outlier
    def __init__(self, contamination=0.1):
        assert 0 <= contamination <= 1
        self.contamination = contamination
        self.split_y = None

    # 计算X对应正态分布的y值
    def normal_distribution(self, X):
        return np.exp(-X ** 2 / 2) / np.sqrt(2 * np.pi)

    # 计算概率密度函数
    def p(self, X):
        p = np.ones(X.shape[0])
        for i in range(X.shape[1]):
            p *= self.normal_distribution(X[:, i])
        return p

    def fit(self, X):
        X_new = self.standard(X)
        y = self.p(X_new)  # 获得正态分布中每一个样本对应的y值
        # 前 split_line 的y值为outlier
        split_line = ceil(self.contamination * X.shape[0])
        self.split_y = y[np.argsort(y)[:split_line][-1]]
        return self

    def predict(self, X):
        predict_value = np.ones(X.shape[0])
        X_new = self.standard(X)
        y = self.p(X_new)
        # 所有小于切分点的y都是outlier
        predict_value[y <= self.split_y] = -1
        return predict_value

得到的效果和EllipticEnvelope一样. 测试代码, 1000个样本, 二者都找到100个异常值

if __name__ == '__main__':
    centers = [[0, 0]]

    ad_model = MyEllipticEnvelope(contamination=0.1)
