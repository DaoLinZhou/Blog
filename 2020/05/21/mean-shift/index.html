Mean shift算法

Mean Shift算法

均值漂移算法.

优点: 不需要告诉它有几个类别


简单来说就是让中心点向更加密集的方向前进

例如此图中, 右下方的数据点更密集, 因此向这个方向移动

也可以理解为每个数据点都在用力拉扯圆心, 最后向合力的方向前进

因此很显然的一个超参数就是圆的半径


对于某一个中心点x, 它的漂移的方向就是:
$$
M_h = \frac 1 n \sum_{x_i \in S_n} (x_i-x)
$$
也就是圆中所有点减去x后的均值, Sn代表到x的距离小于h, 也就是说
$$
S_h(x)={y:(y-x_i)^T(y-x_i) < h^2}
$$
每次新的点就被更新为:
$$
x:=x+M_h
$$

思想:

  1. 随便挑一个没有访问过的点作为起始点(初始时所有点都没有被访问)
  2. 使用mean-shift思想进行移动, 直到收敛
  3. 如果收敛的点与已有的某个中心点过近, 则合并为一个中心点, 反之作为一个独立的中心点
  4. 重复1, 2, 3直到所有点都被访问过
  5. 得到所有的中心点
class MeanShift:
    def __init__(self, bandwidth):
        self.bandwidth = bandwidth
        self.centers_ = []
        self.centers = None
        self.epsilon = bandwidth/10

    def neighbors_indexes(self, point, X):
        # 在X中寻找point为中心, 半径为bandwidth内的数据点
        return np.sum((X - point)**2, axis=1).reshape(-1) <= (self.bandwidth**2)

    # 计算移动的方向
    def shift_vector(self, center, points):
        return np.sum(points - center, axis=0) / points.shape[0]

    # 查看邻近是否有中心点
    def find_same_center(self, center):
        for i, point in enumerate(self.centers_):
            if np.linalg.norm(center-point) <= self.epsilon:
                return i
        return -1

    # 对一个点进行均值漂移
    def _mean_shift_single(self, point, X):
        y = np.zeros(X.shape[0])
        shift_dis = 1
        while shift_dis > 1e-8:
            indexes = self.neighbors_indexes(point, X)
            y[indexes] = 1  # 更新类别
            neighbors = X[indexes]  # 获取邻近节点
            shift = self.shift_vector(point, neighbors)
            shift_dis = shift.dot(shift)
            point = point + shift
        return point, y

    def fit(self, X):
        n_samples, n_features = X.shape
        visited = np.zeros(X.shape[0])

        while np.sum(visited == 0):     # 当还有没被访问过的节点时
            freq = visited == 0
            index = np.random.choice(n_samples, p=freq/np.sum(freq))
            # 对随机选取的没访问过的点进行均值漂移, 飘了飘了[滑稽]
            center, labels = self._mean_shift_single(X[index], X)
            visited += labels
            center_ind = self.find_same_center(center)
            if center_ind == -1:
                self.centers_.append(center)
        self.centers = np.array(self.centers_)

    def predict(self, X):
        n_centers = self.centers.shape[0]
        dist = np.zeros((X.shape[0], n_centers))
        for i in range(0, n_centers):
            dist[:, i] = np.sum((X - self.centers[i])**2, axis=1)
        return np.argmin(dist, axis=1)

但是如果有outlier, 同时epsilon和bandwidth没选好的话, 效果就会较差


但是如果思考一下, 如果根据距离来判断力的大小, 距离越远, 力越小, 对圆的偏移的影响也越小, 这样的话, 是否可以将圆的半径设为无穷大?

就是说每次计算合力的时候都通过所有的样本点进行计算
