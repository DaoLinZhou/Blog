逻辑回归: 解决分类问题


回归问题怎么解决分类问题?

将样本的特征和样本发生的概率联系起来, 换句话说预测的是样本发生的概率是多少, 概率是一个数, 所以称之为回归问题


机器学习算法的本质就是求出一个函数f, 如果有一个x进入就会得到预测值
$$
\hat y=f(x)
$$
这个y其实是我们关心的指标, 如果我们要预测房价, y的值就是房价, 如果要预测成绩, y的值就是成绩

但是在逻辑回归中, 得到的值本质是概率值
$$
\hat p=f(x)
$$
也就是说对于逻辑回归来说, 要得到一个函数f, 来了一个样本放入模型中, f(x) 会计算出一个概率值p, 之后根据概率值p进行分类

如果有50%以上的概率发生的话, 就让y=1, 否则y=0, 当然1和0在具体的问题中可能代表不同的意思

因此, 逻辑回归既可以看作时回归算法, 也可以看作是分类算法. 通常作为分类算法用, 只可以解决二分类问题


线性回归算法中, y的预测值, f(x)=Xbθ 的值域是从负无穷到正无穷

但是概率的值域为[0, 1], 所以直接使用线性回归的方式是不可取的

如果用线性回归的方式: 我们能不能找到一组θ, 使得它和X相乘之后得到的y值来表达事件发生的概率?

单单从应用的角度, 可以这么做, 但是这样做不够好

就是因为概率有值域的限制, 而直接使用线性回归的方式得到的结果没有值域的限制, 使得最终拟合出的线的可信程度非常差

而解决的方案也非常简单

依然是使用线性回归的方式, 依然寻找θ和X进行点乘, 只不过将结果再作为一个特征值, 送给一个σ函数

经过这个σ函数的转换, 转换成一个值域在[0, 1]之间的值

这样就得到了某一个事件的概率


Sigmoid 函数

下面的关键就是Sigma 函数是什么

通常使用一个叫 Sigmoid 的函数

首先尝试绘制一下这个函数, 进而再解释为什么这个函数可以非常好的将一个θX的值转换成概率值

import numpy as np
import matplotlib.pyplot as plt

def sigmoid(t):
    return 1 / (1 + np.exp(-t))

if __name__ == '__main__':
    x = np.linspace(-10, 10, 500)
    y = sigmoid(x)

    plt.plot(x, y)
    plt.show()

可以看到绘制结果就是这样, 这就是sigmoid函数


分析一下sigmoid函数

它的最左端趋近于0, 但达不到0, 最右端趋近于1, 但也达不到1

也就是说这根曲线的值域是在[0, 1]之间, 这就是我们想要的性质
$$
\sigma(t) = \frac 1 {1+e^{-t}}
$$
当 t 取负无穷, 分母就是非常大的值, 式子趋近于0

当 t 取正无穷, 分母趋近于1, 式子也趋近于1

t=0, 式子就是1/2 也就是0.5

之后再来看预估的概率

这样的函数的特点就是把任何数转换到一定范围内 ([0, 1]之间)

而这样的函数还有 tanh, arctan…

剩下的问题就是: 对于给定的样本数据X, y. 我们如何找到参数θ使得用这样的方式, 可以最大程度获得样本数据集X对应的分类输出y?

这是在训练的过程中要作的主要的任务, 也就是拟合训练样本, 要找到一组θ值在训练数据集中有非常好的结果, 即拟合这个样本数据集

逻辑回归这个框架下没有线性回归那么直观, 因为线性回归可以直接使用预测结果减去真值的平方和 (MSE) 来判断拟合的好坏.

但是逻辑回归是一个分类问题, 非常的不一样


逻辑回归的损失函数

怎么建模求出θ?

逻辑回归和线性回归最大的区别, 由于逻辑回归解决的是分类问题, 所以给出来的数据y或者是1, 或者是0.

而我们是根据估计出的 p(代表一个概率), 来决定估计的y是1还是0, 它分成了这两类

相应的损失函数也可以分成这两类

注意y是真值, 也就是说如果真实的y的分类为1的话, 那么估计出来的p越小, yhat就越倾向把值估计成是0, 也就是估计错误, 这种情况下损失就越大

反方向也同理, 如果y=0, 如果给定的样本真实值分类为0的话, p越大, yhat越倾向于分类为1, 损失就越高
