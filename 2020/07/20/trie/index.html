Trie 字典树 前缀树

Trie 字典树

又叫做前缀树.

Trie是一个多叉树


对比Trie和字典


trie是这样的一种数据结构


每一个节点有26(英文有26个字母)个指向下一个节点的指针.

考虑不同语言, 不同情境. 可能不止26个. 例如这里就没有考虑大小写的问题

因此通常我们不指定26个指针, 而是让每个节点有若干个指向下一个节点的指针.

class Node{
    char c;
    Map<char, Node> next;
}

其实我从根节点找下一个节点的过程中, 我们就已经知道字母是谁了

例如我们要搜索cat, 之所以能够来到节点c, 是因为我们在根节点就知道下一个节点应该来到c这个节点.

所以更准确地讲, 应该将字母标在边上

在来到这个节点之前, 我们就知道节点是什么了, 所以这样定义节点是没有问题的

class Node{
    Map<char, Node> next;
}

我们会把叶子节点标记为蓝色. 走到蓝色节点就说明我们找到了一个单词

有些单词可能是另外一些单词的前缀.

例如单词pan和panda, 此时n应该为蓝色, a也应该为蓝色.

所以每一个节点需要一个标识, 来表示是否是某一个单词的结尾

class Node{
    boolean isWord;
    Map<char, Node> next;
}


Trie就像是一个针对字符串的Set.

import java.util.Map;
import java.util.TreeMap;

/**
 * Create by Daolin
 * on 02 / 2019
 */
public class Trie {

    private class Node{
        public boolean isWord;
        public Map<Character, Node> next;

        public Node(boolean isWord){
            this.isWord = isWord;
            next = new TreeMap<>();
        }

        public Node() {
            this(false);
        }
    }

    private Node root;
    private int size;

    public Trie(){
        root = new Node();
        size = 0;
    }

    public int getSize(){
        return size;
    }

    public void add(String word){
        Node cur = root;
        for(int i = 0; i< word.length(); i++){
            char c = word.charAt(i);
            if(cur.next.get(c) == null)
                cur.next.put(c,new Node());
            cur = cur.next.get(c);
        }
        if(!cur.isWord){
            cur.isWord = true;
            size++;
        }
    }

    public boolean remove(String word){
        Node node = search(word);
        if(node == null || !node.isWord)
            return false;
        node.isWord = false;
        size--;
        remove(root, word, 0);
        return true;
    }

    // 返回是否终止remove
    private boolean remove(Node node, String word, int index){
        // 当到达最后一个字母时, 如果后面没有字符, 直接删除, 反之则什么都不做
        if(index == word.length()){
            return !node.next.isEmpty();
        }

        // 从子树中删除单词, 如果不终止remove说明这个字符被真正删去了
        if (!remove(node.next.get(word.charAt(index)), word, index + 1)) {
            node.next.remove(word.charAt(index));
        }

        // 当这个字符是另一个单词的组成部分时终止remove
        return !node.next.isEmpty() || node.isWord;
    }

    public void recursiveAdd(String word){
        add(root,word,0);
    }

    private void add(Node node,String word, int i){
        if(i == word.length()){
            if(!node.isWord){
                node.isWord = true;
                size++;
            }
        }
        char c = word.charAt(i);
        if(node.next.get(c) == null)
            node.next.put(c,new Node());
        add(node.next.get(c), word, i+1);
    }

    public boolean contains(String word){
        Node node = search(word);
        if(node == null)
            return false;
        return node.isWord;
    }

    // 获取单词的叶子节点
    private Node search(String word){
        Node cur = root;
        for (int i = 0; i< word.length(); i++){
            char c = word.charAt(i);
            if(cur.next.get(c) == null)
                return null;
            cur = cur.next.get(c);
        }
        return cur;
    }

    public boolean recursiveContains(String word) {
        return contains(root,word,0);
    }

    private boolean contains(Node node, String word, int i){
        if(i == word.length())
            return node.isWord;
        char c = word.charAt(i);
        if(node.next.get(c) == null){
            return false;
        }
        return contains(node.next.get(c), word, i+1);
    }

    public boolean isPrefix(String prefix){
        Node cur = root;
        for (int i = 0; i < prefix.length(); i++){
            char c = prefix.charAt(i);
            if(cur.next.get(c) == null)
                return false;
            cur = cur.next.get(c);
        }
        return true;
    }

}

Trie的局限性:

需要大量空间!

对于每一个节点, 尽管只表示一个字母, 但是为了向下查找, 它要包含一个map属性.

如果有26个英文字母, 一个单词占用的空间就近乎是自身长度的27倍

为了解决这个问题, 就衍生出很多变种


压缩字典树 Compressed Trie

然而维护的成本更高了.


Ternary Search Trie

如果单词以d开头就从中间查找.

如果单词以小于d的字母开头就向d的左子树进行查找

如果单词以大于d的字母开头就向d的右子树进行查找

这颗树中就存储了dog这个单词


以及后缀树等等…
