从不同的角度, 去看同样的知识, 特别重要

线性映射(linear map)

表示法:

就是线性变换, 把一个n维向量线性变换为m维向量, 实现方法就是矩阵乘法, A是一个m * n的矩阵, n列是因为x是n维, 为了可以相乘, m行是因为变换完成后是m维向量


线性映射的范围(Range of linear map)

假设: 线性变换T是这样

问它的range是什么

首先可以轻易找到这个线性变换对应的矩阵A

即线性变换T的range是它对应的变换矩阵的列空间


正交线性算子(orthogonal linear operators)

定义: 线性变换T:Rn->Rn 是 orthogonal (线性等距 linear isometry) 如果 T保留长度(preserving length)

即: 线性变换后的长度和原来的长度相等


例如旋转变换就是orthogonal linear operators, 因为旋转变换不改变长度


正交矩阵 (orthogonal matrix)

定义: n * n的矩阵A是正交矩阵, 如果ATA=I

性质:

  1. 如果A是正交矩阵则AT也是正交矩阵

  2. 如果A是正交矩阵则A-1也是正交矩阵

  3. A, B都是正交矩阵, 则AB也是正交矩阵

    证明: A-1=AT, B-1=BT, (AB)T =BT AT=B-1A-1=(AB)-1

  4. A是正交矩阵, 则det(A)=1或-1

证明第四条性质:

det(AA-1)=det(I)=1

det(A) * det(A-1) = 1

由于AT=A-1, 所以det(A-1) = det(AT)

由于det(A)=det(A-1)

所以(det(A))2=1

所以det(A)=1或-1


同时这几条定理是等价的:

如果A是n*n的矩阵

值得一提的是第二条和第三条定理

第二条定理就是说 preserving length

第三条定理就是说 preserving dot product

如果从线性变换的视角来看就是, 如果线性变换T保留长度, 则T也保留点乘

可以看到正交算子就是正交矩阵


线性映射的核(kernel)

其实就是线性变换T对应的变换矩阵A的零空间


定理: 如果T是n维到m维的线性变换, 那么T映射n维的子空间到m维子空间

即: T(n维子空间) = m维子空间

证明: 让v是n维空间的子空间

即证明T(v)不为空, T(v)加法封闭, T(v)数量乘法封闭

  1. 不为空, 因为零向量在v中, T(0) = 0所以一定有零向量
  2. 加法封闭
  3. 数量乘法封闭

one to one映射和onto映射

定义one-to-one映射

T(x)=T(y)当且仅当仅当x=y时成立

定义onto映射

T是onto如果range(T) = codomain of T = Rm


定理1

定理: T是one-to-one映射当且仅当ker(T)={0}

证明 forward:

假设ker(T) != 0, 说明非零向量u在ker(T)中, 换句话说u在A的零空间中, Au=0, A0=0, 有两个向量经过T映射指向同一个向量, 所以T不是one-to-one

证明 backward:

假设 T不是one-to-one, 则有T(u)=T(v), u!=v

T(u)-T(v)=0 因此 T(u-v)=0. 因为u不等于v所以ker(T)中包含非零向量u-v, 矛盾


ker(T)={0}说明T对应的变换矩阵A的零空间只有零向量, 即Au=0有唯一解

对于一般矩阵来说, Au=0有唯一解那么A的行数一定大于等于列数

且rank(A) = 列数, 换句话说所有列向量线性无关

也就是说当A所有列线性无关时T是one-to-one映射

即A的行最简形式为


定理2

定理: T是onto当且仅当Ax=b有解对于所有m维向量b, 即A的列空间是Rm

A是mn的矩阵, A的列空间是Rm, 说明A的列数n>=A的行数m, 且*A的列向量生成了整个m维空间**,

即A的行最简形式应该长这样:


A所有列线性无关时T是one-to-one映射

A的列向量生成了整个m维空间时T是onto

而当A是n*n的矩阵时, 如果A代表one-to-one, 则列向量线性无关, A可逆, 因此A的列向量生成n维空间, 所以A也代表onto. 对于方阵A来说, 它是one-to-one 当且仅当 它是onto

而且如果一个mn的矩阵A是one-to-one, 那么A的列向量线性无关, 因此m>=n, 如果A是onto, 则A的列向量生成m维空间, n>=m, 因此矩阵A如果既是one-to-one, 又是onto, 那么 m>=n ^ n>=m, 因此m=n, A一定是方阵, 而且由于A的列向量线性无关, *A一定可逆**


列空间的理解

一个矩阵的列空间是这个矩阵列向量的线性组合, 然而当我们用列视角看矩阵乘法时

Ax=b 这个式子中, b就是A的列向量的线性组合, 因此所谓的列空间可以理解为Ax=b中, b所有可能值的集合

因此, 如果Ax=b这个系统有解, 当且仅当( if and only if ) 向量b在A的列空间中.

反之如果Ax=b没有解, b就不在A的列空间中
