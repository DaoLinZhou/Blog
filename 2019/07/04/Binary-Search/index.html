如何在顺序数组中查找某一个元素的位置

二分搜索法就是很好的算法

我将实现递归和非递归两个版本的二分搜索法

递归版本思维起来更容易, 但性能比循环差, 但它们都是log(n)级别的算法

基本思想就是找到数组中间的元素的值,

由于数组是顺序的, 所以如果目标值小于中间值, 那么它就应该在 [l…mid-1] 之中

​ 目标值大于中间值, 那么它就应该在 [mid+1…r] 之中


循环版本

值得一提的是二分查找的一个容易被忽视的bug

如果 l 和 r 都非常大, l+r 可能会溢出, 变为负数, 而此时的mid就会是错误的值

所以求mid的时候应该用减法来替代加法

//二分查找法, 在有序数组中查找target
//经典循环实现
template <typename T>
int binarySearchLoop(T arr[], int n, T target){
    int l=0, r=n-1;        // 在[l...r]的范围内寻找target
    while(l <= r){        // 重新上方看定义, l==r时[l...r]范围依然有效
//        int mid = (l+r)/2;
        int mid = l + (r-l)/2;
        if(arr[mid] > target)
            r = mid-1;
        else if(arr[mid] < target)
            l = mid+1;
        else
            return mid;
    }
    return -1;
}

递归版本

template <typename T>
int _binarySearch(T arr[], int l, int r, T target){
    if(l > r)
        return -1;
    int mid = l+(r-l)/2;
    if(arr[mid] > target)
        return _binarySearch(arr, l, mid-1, target);
    else if(arr[mid] < target)
        return _binarySearch(arr, mid+1, r, target);
    else
        return mid;
}

template <typename T>
int binarySearch(T arr[], int n, T target){
    return _binarySearch(arr, 0, n-1, target);
}

都比较简单, 没什么好解释的.


二分查找的变种

上面的两个版本都只是找到就停止查找, 没有考虑数组中有相同元素的可能性.

而且如果没有元素, floor 和 ceil也不应该返回-1, 而是:

思想依然简单, 如果找到了, 那么就 向前(floor) 或 向后(ceil) 遍历. 如果没找到, 此时r < l, 那么 r 对应的就是floor, l 对应的就是ceil.

floor和ceil大致思路是一样的, 下面我只给出floor和ceil的循环版本实现. 有兴趣的胖友可以试着实现递归版本

//如果查找值比arr[0]还要小时, 返回-1
template <typename T>
int floor(T arr[], int n, T target){

    assert( n >= 0 );

    int l=0, r=n-1;
    while(l <= r){
        int mid = l + (r-l)/2;
        if(arr[mid] > target)
            r = mid-1;
        else if(arr[mid] < target)
            l = mid+1;
        else{
            // 找到了, 开始向前查找
            for(int i=mid-1; i>=l; i--) {
                if (arr[i] != target)
                    return i + 1;
                // 如果是第一个元素, 且arr[l] == target
                if (i == l)
                    return l;
            }
        }
    }
    return r;
}

//如果查找的值比arr[n-1] 还大时, 返回n
template <typename T>
int ceil(T arr[], int n, T target){

    assert( n >= 0 );

    int l=0, r=n-1;
    while(l <= r){
        int mid = l + (r-l)/2;
        if(arr[mid] > target)
            r = mid-1;
        else if(arr[mid] < target)
            l = mid+1;
        else{
            // 找到了, 开始向后查找
            for(int i=mid+1; i<=r; i++) {
                if (arr[i] != target)
                    return i - 1;
                //如果是最后一个元素, 且 arr[r] == target
                if (i == r)
                    return r;
            }
        }
    }
    return l;
}
