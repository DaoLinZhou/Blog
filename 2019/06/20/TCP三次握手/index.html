TCP зҡ„дёүж¬ЎжҸЎжүӢ

еҪ“еә”з”ЁзЁӢеәҸеёҢжңӣйҖҡиҝҮTCPдёҺеҸҰдёҖдёӘеә”иҜҘзЁӢеәҸйҖҡдҝЎж—¶, еҲҷдјҡеҸ‘йҖҒдёҖдёӘйҖҡдҝЎиҜ·жұӮ, иҝҷдёӘиҜ·жұӮеҝ…йЎ»иў«йҖҒеҲ°дёҖдёӘзЎ®еҲҮзҡ„ең°еқҖ.

еңЁеҸҢж–№жҸЎжүӢд№ӢеҗҺ, TCP е°ҶеңЁдёӨдёӘеә”з”Ёд№Ӣй—ҙе»әз«ӢдёҖдёӘе…ЁеҸҢе·ҘпјҲFull Duplexпјүзҡ„йҖҡдҝЎ. иҝҷе°ҶеҚ з”ЁдёӨдёӘи®Ўз®—жңәд№Ӣй—ҙзҡ„йҖҡдҝЎзәҝи·Ҝ. зӣҙеҲ°е®ғиў«дёҖж–№ жҲ– еҸҢж–№е…ій—ӯдёәжӯў


д»Җд№ҲжҳҜе…ЁеҸҢе·Ҙ (Full Duplex)

и®Ўз®—жңә A з»ҷ B еҸ‘йҖҒдҝЎжҒҜ, еңЁеҸ‘йҖҒдҝЎжҒҜзҡ„еҗҢж—¶, B д№ҹиғҪз»ҷ A еӣһеҸ‘дҝЎжҒҜ


жөҒзЁӢеӣҫ

еҸҜд»ҘзңӢеҲ° жңҚеҠЎеҷЁеҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒack е°ұжҳҜ SYN е’Ң SEQ зҡ„е’Ң (x+1), еҗҢж—¶иҮӘе·ұд№ҹиҰҒеҲқе§ӢеҢ–дёҖдёӘеәҸеҲ—еҸ·

иҖҢ е®ўжҲ·з«Ҝ еҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒзҡ„з¬¬дәҢж¬ЎиҜ·жұӮдёӯ, seqе°ұзӯүдәҺдёҠж¬Ўж”¶еҲ°зҡ„ackзҡ„еҖј, е№¶дё”жҗәеёҰзҡ„ackдёәy+1, еҚіжңҹжңӣдёӢдёҖж¬ЎжңҚеҠЎеҷЁеҸ‘йҖҒзҡ„seq


жҠ“еҢ…

еҮҶеӨҮе·ҘдҪң: иҺ·еҸ–Bз«ҷзҡ„ipең°еқҖ

ж—ўз„¶иҰҒжҠ“еҢ…, йҰ–е…ҲиҰҒзҹҘйҒ“иҰҒжҠ“ вҖңи°ҒвҖқ зҡ„еҢ…, йҖҡиҝҮ ping е‘Ҫд»ӨиҺ·еҸ– Bз«ҷ ip

жӯЈејҸејҖе§Ӣ

еҗҜеҠЁ Wireshark йҖүжӢ©WALN, е№¶дё”ејҖеҗҜжҚ•иҺ· еҲ·ж–°bз«ҷйЎөйқў

и®ҫзҪ®иҝҮж»Өи§„еҲҷ:

вҖӢ жҲ‘д»¬еҸӘе…іжіЁ ip дёә 148.153.38.118 зҡ„зӣёеә”е’ҢиҜ·жұӮ

з»“жһң

148.153.38.118 жҳҜbз«ҷзҡ„ ip, йӮЈд№Ҳжң¬жңәзҡ„ipдҫҝдёә 172.29.154.49

и§Јжһҗ

дёӢйқўзҡ„и§ЈжһҗиҝҮзЁӢжңҖеҘҪй…ҚеҗҲжөҒзЁӢеӣҫи§ӮзңӢ

йҰ–е…Ҳжң¬жңәеҗ‘Bз«ҷеҸ‘йҖҒдёҖдёӘSYNеҢ…, seq (д»Һжң¬жңәеҫҖBз«ҷзҡ„seq) =0,

д№ӢеҗҺbз«ҷеӣһеӨҚжң¬жңәдёҖдёӘ(SYN,ACK)еҢ… ack = 0+1 = 1, seq (д»ҺBз«ҷеҫҖжң¬жңәзҡ„seq) = 0

жңҖеҗҺжң¬жңәз»ҷBз«ҷеӣһеӨҚдёҖдёӘACKеҢ… seq = (bз«ҷз»ҷжҲ‘д»¬зҡ„ack) = 1, ack = seq + SYN = 0 + 1 = 1;

дёҠйқўиҝҷдәӣжӯҘйӘӨе°ұжҳҜTCPзҡ„дёүж¬ЎжҸЎжүӢ


дёәд»Җд№ҲиҰҒдёүж¬ЎжҸЎжүӢ?

дёәдәҶеҲқе§ӢеҢ–Sequence Number зҡ„еҲқе§ӢеҖј, еҸҢж–№иҰҒйҖҡзҹҘеҜ№ж–№иҮӘе·ұзҡ„secquence Number.

иҝҷдёӘеҸ·дјҡдҪңдёәд»ҘеҗҺйҖҡи®Ҝзҡ„еәҸеҸ·, д»ҘдҝқиҜҒж”¶еҲ°зҡ„ж•°жҚ®дёҚдјҡеӣ дёәзҪ‘з»ңдј иҫ“й—®йўҳиҖҢд№ұеәҸ (TCP дјҡз”ЁиҝҷдёӘеәҸеҸ·жқҘжӢјжҺҘж•°жҚ®)

жңҖеҗҺдёҖж¬Ўжң¬жңәеҸ‘еҫҖBз«ҷзҡ„ACKеҢ…, жҳҜе‘ҠиҜүжңҚеҠЎеҷЁвҖқжң¬жңәж”¶еҲ°дҪ зҡ„seq numberдәҶвҖқ


д»ҺдёҖдёӘж•…дәӢиҜҙиө·

жҲ‘и®°еҫ—иҮӘе·ұд»ҘеүҚзңӢиҝҮдёҖдёӘж•…дәӢ, жҸҸиҝ°зҡ„е°ұжҳҜTCPдёәд»Җд№ҲиҰҒжҸЎжүӢ3ж¬Ў, дҪҶжҳҜеҰӮд»Ҡе·Із»ҸжүҫдёҚеҲ°еҺҹж–ҮдәҶ, жүҖд»ҘеҸӘиғҪжҠҠжҲ‘и®°еҝҶдёӯзҡ„ж•…дәӢеӨ§жҰӮи®Іиҝ°дёҖйҒҚ.

ж•…дәӢ

зҺ°еңЁжңүдёүдёӘеӣҪе®¶, A B C.

AиҰҒиҒ”еҗҲBдёҖиө·ж”»жү“C, дәҺжҳҜйҖҒдҝЎз»ҷBеӣҪ, е‘ҠиҜүд»–XXXж—¶й—ҙ, дҪ жҲ‘дёӨеӣҪдёҖиө·ж”»жү“CеӣҪ.

дҪҶй—®йўҳжҳҜ, дҝЎдҪҝйҖҒдҝЎеҸҜиғҪдјҡиў«CеӣҪжҲӘиҺ·, еҜјиҮҙBдёҚзҹҘйҒ“иҰҒж”»жү“CеӣҪ, иҖҢAеӣҪеҚҙеңЁзәҰе®ҡзҡ„ж—¶й—ҙеӮ»fufuең°еҶІдёҠеҺ»дәҶ.

жүҖд»ҘBеӣҪиҰҒз»ҷAеӣҪеӣһдёӘдҝЎ, е‘ҠиҜүд»–, вҖңжҲ‘зҹҘйҒ“дәҶ, еңЁXXXж—¶й—ҙ, е’ұдҝ©дёҖиө·иҝӣж”»CеӣҪвҖқ

й—®йўҳеҸҲжқҘдәҶ, дҝЎдҪҝйҖҒдҝЎеҸҜиғҪдјҡиў«CеӣҪжҲӘиҺ·, BеӣҪзҹҘйҒ“иҰҒеңЁXXXж—¶й—ҙж”»жү“CеӣҪ, дҪҶAдёҚзҹҘйҒ“BзҹҘйҒ“(жңүзӮ№з»•еҸЈ),

A жІЎж”¶еҲ°дҝЎд»¶, Aе°ұжҖҖз–‘, жҳҜдёҚжҳҜиҮӘе·ұдёҠж¬ЎеҸ‘зҡ„дҝЎиў«жҲӘиҺ·дәҶ, иҖҢBдёҚзҹҘйҒ“жңүиҝҷд№ҲеӣһдәӢ. жүҖд»ҘдёҺе…¶еӮ»fufuзҡ„ж”»жү“CеӣҪ, AдјҡжҢүе…өдёҚеҠЁ. иҖҢBеҸҲеӮ»fufuең°ж”»жү“CеӣҪеҺ»дәҶ.

жӯӨж—¶AеҰӮжһңж”¶еҲ°дҝЎд»¶, иҰҒз»ҷBеӣһдҝЎ: вҖңжҲ‘зҹҘйҒ“дҪ зҹҘйҒ“дәҶ, еҲ°ж—¶еҖҷдёҖиө·еҺ»е№Ід»–вҖқ

еҰӮжһңиҝҷд№ҲжҢҒз»ӯдёӢеҺ», A дёҺ BиҰҒж— йҷҗең°з»ҷеҜ№ж–№еҸ‘йҖҒдҝЎд»¶. дәҺжҳҜ, дәәд»¬(и®Ўз®—жңәз§‘еӯҰе®¶д»¬)и®Өдёә, жҲҗеҠҹдј иҫ“3ж¬Ў, е°ұеҸҜд»ҘиҜҒжҳҺдә’зӣёд№Ӣй—ҙжҳҜеҸҜд»ҘйҖҡдҝЎзҡ„


йҰ–ж¬ЎжҸЎжүӢзҡ„йҡҗжӮЈ

й—®йўҳзҡ„иө·еӣ 

Server ж”¶еҲ° Client зҡ„SYN, еӣһеӨҚSYN-ACK жІЎжңүж”¶еҲ°ACKзЎ®и®Ө

жӯӨж—¶иҝһжҺҘеӨ„дәҺдёӯй—ҙзҠ¶жҖҒ, жІЎжңүжҲҗеҠҹ, д№ҹжІЎжңүеӨұиҙҘ

Server дёҚж–ӯйҮҚиҜ•зӣҙеҲ°и¶…ж—¶, Linux й»ҳи®ӨйҮҚиҜ•5ж¬Ў, йҮҚиҜ•й—ҙйҡ”д»Һ1з§’ејҖе§Ӣ, жҜҸж¬Ўзҝ»еҖҚ, (1, 2, 4, 8, 16) еңЁз¬¬дә”ж¬ЎеҸ‘еҮәеҺ»д№ӢеҗҺ, дјҡзӯүеҫ…32 з§’, дёҖе…ұдјҡзӯүеҫ…63з§’жүҚеҲӨе®ҡдёәи¶…ж—¶,ж–ӯејҖиҝһжҺҘ


еӣ жӯӨеҸҜиғҪдјҡжңүSYN Floodж”»еҮ»зҡ„йЈҺйҷ©

жҒ¶ж„Ҹе®ўжҲ·з«Ҝдјҡз»ҷжңҚеҠЎеҷЁеҸ‘йҖҒдёҖдёӘSYNжҠҘж–Ү, д№ӢеҗҺд»–е°ұдёӢзәҝдәҶ,

иҖҢжңҚеҠЎеҷЁйңҖиҰҒй»ҳи®Өзӯү 63 з§’жүҚдјҡж–ӯејҖиҝһжҺҘ. ж”»еҮ»иҖ…йҮҚеӨҚеӨҡж¬Ў, дҫҝжҠҠ SYN иҝһжҺҘйҳҹеҲ—иҖ—е°Ҫ, жӯЈеёёиҜ·жұӮж— жі•еӨ„зҗҶ


йҳІиҢғжҺӘж–Ҫ

SYN йҳҹеҲ—ж»ЎеҗҺ, йҖҡиҝҮtcp_syncookiesеҸӮж•°еӣһеҸ‘SYN cookies

йҖҡиҝҮ еҺҹең°еқҖз«ҜеҸЈ, зӣ®ж Үең°еқҖз«ҜеҸЈ, ж—¶й—ҙжҲі жү“йҖ еҮәдёҖдёӘзү№еҲ«зҡ„seq number з®Җз§°SYN cookie.

еҰӮжһңжҳҜж”»еҮ»иҖ…, з”ұдәҺе®ғдёӢзәҝдәҶ, жүҖд»ҘдёҚдјҡжңүд»»дҪ•еҸҚеә”.

иҖҢжӯЈеёёиҝһжҺҘеҲҷдјҡжҠҠSYN cookies еҸ‘еӣһжқҘ, жңҚеҠЎз«ҜйҖҡиҝҮSYN cookieе»әз«ӢиҝһжҺҘ

е»әз«ӢиҝһжҺҘеҗҺ, ClientеҮәзҺ°ж•…йҡңжҖҺд№ҲеҠһ

дҝқжҙ»жңәеҲ¶

еңЁдёҖж®өж—¶й—ҙ(дҝқжҙ»ж—¶й—ҙ keep alive time) иҝһжҺҘеӨ„дәҺйқһжҙ»еҠЁзҠ¶жҖҒ,

ејҖеҗҜдҝқжҙ»зҡ„дёҖз«Ҝе°Ҷеҗ‘еҜ№ж–№еҸ‘йҖҒдҝқжҙ»жҺўжөӢжҠҘж–Ү, еҰӮжһңжңӘж”¶еҲ°зӣёеә”еҲҷз»§з»ӯеҸ‘йҖҒ

зӣҙеҲ°е°қиҜ•ж¬Ўж•°иҫҫеҲ°дҝқжҙ»жҺўжөӢж•°д»ҚжңӘж”¶еҲ°зӣёеә”еҲҷдёӯж–ӯиҝһжҺҘ, и®ӨдёәеҜ№ж–№дё»жңәдёәдёҚеҸҜиҫҫ
