TCPзҡ„ж»‘еҠЁзӘ—еҸЈеҸҜд»ҘеҜ№жөҒйҮҸиҝӣиЎҢжҺ§еҲ¶, жқҘйҒҝе…ҚжӢҘеЎһзҡ„еҸ‘з”ҹ, жң¬ж–Үдјҡи®Іиҝ°дёҖдёӢж»‘еҠЁзӘ—еҸЈзҡ„еӨ§иҮҙеҺҹзҗҶ

RTT е’Ң RTO

RTT : еҸ‘йҖҒдёҖдёӘж•°жҚ®еҢ…еҲ°ж”¶еҲ°еҜ№еә”зҡ„ACK, жүҖиҠұиҙ№зҡ„ж—¶й—ҙ

RTO : йҮҚдј ж—¶й—ҙй—ҙйҡ”, еҰӮжһңж”¶еҲ°ACK е°ұдёҚдјҡйҮҚдј , RTO жҳҜйҖҡиҝҮ RTT и®Ўз®—еҮәжқҘзҡ„


TCP зҡ„ж»‘еҠЁзӘ—еҸЈ

йҰ–е…ҲжҲ‘д»¬зҹҘйҒ“ TCP дјҡжҠҠж•°жҚ®жӢҶеҲҶжҲҗж®өиҝӣиЎҢеҸ‘йҖҒ.

дҪҶжҳҜдёҚеҸҜиғҪдёҖж®өдёҖж®өең°иҝӣиЎҢеҸ‘йҖҒ, зӯүеҲ°дёҠдёҖж®өж•°жҚ®иў«зЎ®и®Өд№ӢеҗҺжүҚеҸ‘йҖҒдёӢдёҖж®өж•°жҚ®, жүҖд»Ҙеҝ…йЎ»жү№йҮҸдј иҫ“, е№¶дё” TCP иҰҒи§ЈеҶіеҸҜйқ дј иҫ“, еҢ…д№ұеәҸзҡ„й—®йўҳ, жүҖд»ҘTCPиҰҒзҹҘйҒ“зҪ‘з»ңе®һйҷ…еӨ„зҗҶж•°жҚ®зҡ„еёҰе®Ҫ, жҲ–жҳҜе®һйҷ…еӨ„зҗҶйҖҹеәҰ, иҝҷж ·жүҚдёҚдјҡеј•иө·зҪ‘з»ңжӢҘеЎһиҖҢеҜјиҮҙдёўеҢ….


TCPдҪҝз”Ёж»‘еҠЁзӘ—еҸЈеҒҡжөҒйҮҸжҺ§еҲ¶дёҺд№ұеәҸйҮҚжҺ’

  • дҝқиҜҒTCPзҡ„еҸҜйқ жҖ§
  • дҝқиҜҒTCPзҡ„жөҒжҺ§зү№жҖ§

TCPзҡ„жҠҘж–Үж јејҸдёӯжңүwindow, еҚі:зӘ—еҸЈ

е®ғзҡ„дҪңз”ЁжҳҜ: жҺҘеҸ—ж–№йҖҡзҹҘеҸ‘йҖҒж–№иҮӘе·ұиҝҳжңүеӨҡе°‘зј“еҶІеҢәеҸҜд»ҘжҺҘеҸ—ж•°жҚ®

иҖҢеҸ‘йҖҒж–№ж №жҚ®жҺҘж”¶ж–№зҡ„еӨ„зҗҶиғҪеҠӣиҖҢеҸ‘йҖҒж•°жҚ®, дёҚеҜ№еҜјиҮҙжҺҘеҸ—ж–№еӨ„зҗҶдёҚиҝҮжқҘ


зӘ—еҸЈж•°жҚ®зҡ„и®Ўз®—иҝҮзЁӢ

е·Ұиҫ№жҳҜTCPеҚҸи®®еҸ‘йҖҒз«Ҝзҡ„зј“еҶІеҢә

еҸіиҫ№жҳҜжҺҘж”¶з«Ҝзј“еҶІеҢә

е·Ұиҫ№еҫҖеҸіиҫ№еҸ‘ж•°жҚ®


е·Ұиҫ№еҸ‘йҖҒж–№:

вҖӢ еҜ№дәҺеҸ‘йҖҒж–№ LastByteAcked жҢҮеҗ‘ж”¶еҲ°зҡ„иҝһз»ӯжңҖеӨ§ACKзҡ„дҪҚзҪ®. д№ҹе°ұжҳҜзӣ®еүҚе·Із»Ҹж”¶еҲ°ACKеҲ°иҝҷдёӘдҪҚзҪ®. LastByteSent жҢҮеҗ‘е·ІеҸ‘йҖҒзҡ„жңҖеҗҺдёҖдёӘеӯ—иҠӮзҡ„дҪҚзҪ®, иҜҘдҪҚзҪ®еҸӘжҳҜеҸ‘еҮәеҺ»дәҶ, дҪҶиҝҳжІЎжңүж”¶еҲ°ACK. LastByteWritten жҢҮеҗ‘дёҠеұӮеә”з”Ёд»ҘеҶҷе®Ңзҡ„жңҖеҗҺдёҖдёӘдҪҚзҪ®, еҚі: еҪ“еүҚзЁӢеәҸ, е·Із»ҸеҮҶеӨҮеҶҷеҘҪйңҖиҰҒеҸ‘йҖҒзҡ„жңҖж–°зҡ„ж•°жҚ®.

вҖӢ LastByteAcked д№ӢеүҚ: еҸ‘йҖҒе№¶ж”¶еҲ°ACK

вҖӢ LastByteAcked - LastByteSent : еҸ‘йҖҒ, дҪҶжІЎжңүжҺҘеҸ—еҲ°ACK

вҖӢ LastByteSent - LastByteWritten : еҫ…еҸ‘йҖҒзҡ„ж•°жҚ®

вҖӢ LastByteAcked еҲ° LastByteWritten д№Ӣй—ҙжҳҜиҝһз»ӯзҡ„, жІЎжңүй—ҙйҡ”зҡ„


еҸіиҫ№жҺҘж”¶ж–№:

LastByteRead жҢҮеҗ‘дёҠеұӮеә”з”Ёе·Із»ҸиҜ»е®Ңзҡ„жңҖеҗҺдёҖдёӘеӯ—иҠӮзҡ„дҪҚзҪ®

NextByteExpected жҢҮеҗ‘ж”¶еҲ°зҡ„иҝһз»ӯжңҖеӨ§зҡ„sequenceзҡ„дҪҚзҪ®

LastByteRcvd жҢҮеҗ‘е·Іж”¶еҲ°жңҖеҗҺдёҖдёӘеӯ—иҠӮзҡ„дҪҚзҪ®

NextByteExpected е’Ң LastByteRcvd д№Ӣй—ҙжңүдёҖдәӣ seq е№¶жІЎжңүеҲ°иҫҫ(з©әзҷҪ)


и®Ўз®—EffectiveWindow

AdvertisedWindow жҲ‘иғҪеӨҹжҺҘеҸ—зҡ„зӘ—еҸЈ(жҺҘж”¶ж–№иҝҳиғҪеӨ„зҗҶж•°жҚ®зҡ„йҮҸ)

MaxRcvBuffer жҺҘж”¶ж–№иғҪжҺҘеҸ—зҡ„жңҖеӨ§ж•°жҚ®йҮҸ(жҺҘж”¶з«Ҝзј“еӯҳж—¶зҡ„еӨ§е°Ҹ)

(LastByteRcvd - LastByteRead) жҳҜдёәе·Іж”¶еҲ°зҡ„ж•°жҚ®е’ҢжңӘж”¶еҲ°зҡ„ж•°жҚ®з•ҷеҮәзҡ„з©әй—ҙ


д№ӢеҗҺжҠҠ AdvertisedWindow еҸ‘йҖҒ(е‘ҠзҹҘ) з»ҷ еҸ‘йҖҒз«Ҝ.

еҸ‘йҖҒз«ҜиҰҒдҝқиҜҒLastByteAcked - LastByteSentдёӯй—ҙзҡ„ж•°жҚ®йҮҸиҰҒе°ҸдәҺAdvertisedWindow . еҚі: е·ІеҸ‘йҖҒдҪҶжІЎе“Қеә”зҡ„ж•°жҚ®йҮҸиҰҒе°ҸдәҺжҺҘж”¶ж–№иҝҳиғҪеӨ„зҗҶж•°жҚ®зҡ„йҮҸ


LastByteSent - LastByteAcked дёәе·ІеҸ‘йҖҒзҡ„ж•°жҚ®йҮҸ

AdvertisedWindow жҺҘж”¶ж–№иҝҳиғҪеӨ„зҗҶж•°жҚ®зҡ„йҮҸ

EffectiveWindow иҝҳиғҪеҸ‘йҖҒзҡ„ж•°жҚ®йҮҸ


TCPдјҡиҜқеҸ‘йҖҒж–№

TCPдјҡиҜқеҸ‘йҖҒж–№, еңЁд»»дҪ•ж—¶еҖҷ, зј“еӯҳйғҪеҸҜд»ҘеҲҶдёә4зұ»

Category #1 : е·Із»ҸеҸ‘йҖҒе№¶дё”еҫ—еҲ°з«Ҝзҡ„еӣһеә”зҡ„, еҚі: LastByteAcked д№ӢеүҚ

Category #2 : е·Із»ҸеҸ‘йҖҒдҪҶиҝҳжІЎж”¶еҲ°з«Ҝзҡ„еӣһеә”зҡ„, еҚі: LastByteAcked - LastByteSent

Category #3 : жңӘеҸ‘йҖҒдҪҶеҜ№з«Ҝе…Ғи®ёеҸ‘йҖҒзҡ„, еҚі: LastByteSent - LastByteWritten

Category #4 : жңӘеҸ‘йҖҒ, дё”з”ұдәҺиҫҫеҲ°windowзҡ„еӨ§е°ҸдёҚе…Ғи®ёеҸ‘йҖҒзҡ„ж•°жҚ®


иҖҢCategory #2, #3 жүҖз»„жҲҗзҡ„иҝһз»ӯз©әй—ҙ, еҲҷиў«з§°дёәеҸ‘йҖҒзӘ—еҸЈ

зӘ—еҸЈж»‘еҠЁ

еҪ“ж”¶еҲ°жҺҘж”¶ж–№ж–°зҡ„ACK(еҜ№дәҺеҗҺз»ӯзӘ—еҸЈеӯ—иҠӮзҡ„зЎ®и®Ө)ж—¶, зӘ—еҸЈиҝӣиЎҢж»‘еҠЁ

еҰӮеӣҫжүҖзӨә, еҰӮжһң32, 33 жІЎиў«зЎ®и®Ө, дҪҶ34 иў«зЎ®и®ӨдәҶ, зӘ—еҸЈд№ҹдёҚдјҡеҗ‘еҸіж»‘еҠЁ

еҸӘжңүеҪ“иҝһз»ӯиў«зЎ®и®Өд№ӢеҗҺ(32, 33, 34 вҖҰ), зӘ—еҸЈжүҚдјҡж»‘еҠЁ, иҖҢжІЎж»‘еҠЁд»ҘеүҚ, 52 еҚід№ӢеҗҺзҡ„ж•°жҚ®жҳҜдёҚиғҪиў«еҸ‘йҖҒзҡ„.

еҒҮи®ҫеҲ°35 йғҪиў«зЎ®и®ӨдәҶ, йӮЈд№ҲзӘ—жҲ·ж»‘еҠЁ4дҪҚ, еҸҳжҲҗд»Һ36ејҖе§Ӣ, иҝӣиҖҢе°ұиғҪеҸ‘йҖҒ52 - 55 зҡ„ж•°жҚ®дәҶ


TCPдјҡиҜқжҺҘж”¶ж–№

еҜ№дәҺжҺҘж”¶ж–№, жңүдёүз§ҚзҠ¶жҖҒ:

Category #1 + 2 : е·ІжҺҘж”¶, е№¶дё”е·Із»ҸеҸ‘йҖҒеӣһжү§

Category #3 : жңӘжҺҘж”¶, дҪҶеҸҜд»ҘжҺҘж”¶

Category #4 : жңӘжҺҘж”¶, дҪҶз”ұдәҺиҫҫеҲ°зӘ—еҸЈзҡ„йҳҲеҖј, жүҖд»ҘдёҚиғҪжҺҘж”¶


й»ҳи®ӨжІЎжңүеә”з”Ёе»¶иҝҹ, дёҚеӯҳеңЁе·ІжҺҘж”¶дҪҶжңӘеӣһеӨҚзҡ„зҠ¶жҖҒ

е…¶дёӯ Category #3 (жңӘжҺҘж”¶, дҪҶеҸҜд»ҘжҺҘж”¶) е°ұжҳҜжҺҘж”¶зӘ—еҸЈ

жҺҘж”¶ж–№зҡ„ж»‘еҠЁзӘ—еҸЈе’ҢеҸ‘йҖҒж–№зҡ„дёҖиҮҙ, еҸӘжңүж”¶еҲ°иҝһз»ӯзҡ„ackжүҚдјҡз§»еҠЁе·Ұиҫ№з•Ң,

еҰӮжһңеүҚйқўжІЎжҺҘж”¶, иҖҢеҗҺйқўжҺҘж”¶, еҲҷдёҚдјҡж»‘еҠЁзӘ—еҸЈ, зЎ®дҝқж•°жҚ®дјҡиў«йҮҚдј .


жіЁж„Ҹ:

ж»‘еҠЁзӘ—еҸЈзҡ„еӨ§е°ҸеҸҜд»Ҙж №жҚ®дёҖе®ҡзӯ–з•ҘеҠЁжҖҒи°ғж•ҙ, еә”з”Ёдјҡж №жҚ®иҮӘиә«еӨ„зҗҶиғҪеҠӣзҡ„еҸҳеҢ–йҖҡиҝҮжң¬ж®өTCPзӘ—еҸЈзҡ„еӨ§е°Ҹзҡ„жҺ§еҲ¶, жқҘе®һзҺ°еҜ№з«Ҝзҡ„еҸ‘йҖҒзӘ—еҸЈиҝӣиЎҢжөҒйҮҸжҺ§еҲ¶
