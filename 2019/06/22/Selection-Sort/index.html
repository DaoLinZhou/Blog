жҖ»з»“:

з©әй—ҙеӨҚжқӮеәҰ: O(1)

зЁіе®ҡжҖ§: дёҚзЁіе®ҡ

зү№зӮ№: дәӨжҚўж¬Ўж•°жңҖе°‘зҡ„жҺ’еәҸ

жҖқжғі: йҒҚеҺҶnеҸҳж•°з»„, жҜҸдёҖж¬ЎжүҫеҪ“еүҚд№ұеәҸйғЁеҲҶжңҖе°Ҹзҡ„е…ғзҙ , жҠҠе®ғе’Ңд№ұеәҸйғЁеҲҶз¬¬дёҖдёӘе…ғзҙ дә’жҚў


дёҫдёӘж —еӯҗ

еҰӮжһңж•°з»„зҡ„е…ғзҙ дёә {3, 8, 2, 5, 1, 4, 7}

еҲҷйҰ–е…ҲйҒҚеҺҶдёҖйҒҚж•°з»„, еҜ»жүҫжңҖе°Ҹе…ғзҙ , жңҖе°Ҹе…ғзҙ дёә 1, жҠҠ 1 е’Ң 3 дәӨжҚўдҪҚзҪ®

з¬¬дёҖж¬ЎеҫӘзҺҜ: {1, 8, 2, 5, 3, 4, 7} жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1} д№ұеәҸйғЁеҲҶдёә {8, 2, 5, 3, 4, 7}

з¬¬дәҢж¬ЎеҫӘзҺҜ: {1, 2, 8, 5, 3, 4, 7} дә’жҚў2, 8 зҡ„дҪҚзҪ®, жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1, 2} д№ұеәҸйғЁеҲҶдёә {8, 5, 3, 4, 7}

з¬¬еӣӣж¬ЎеҫӘзҺҜ: {1, 2, 3, 4, 8, 5, 7} дә’жҚў4, 5 зҡ„дҪҚзҪ®, жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1, 2, 3, 4} д№ұеәҸйғЁеҲҶдёә {8, 5, 7}

з¬¬дә”ж¬ЎеҫӘзҺҜ: {1, 2, 3, 4, 5, 8, 7} дә’жҚў5, 8 зҡ„дҪҚзҪ®, жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1, 2, 3, 4, 5} д№ұеәҸйғЁеҲҶдёә {8, 7}

з¬¬е…ӯж¬ЎеҫӘзҺҜ: {1, 2, 3, 4, 5, 7, 8} дә’жҚў5, 8 зҡ„дҪҚзҪ®, жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1, 2, 3, 4, 5, 7} д№ұеәҸйғЁеҲҶдёә {8}

з¬¬дёғж¬ЎеҫӘзҺҜ: {1, 2, 3, 4, 5, 7, 8} дә’жҚў8, 8 зҡ„дҪҚзҪ®, жҺ’еҘҪеәҸзҡ„йғЁеҲҶдёә {1, 2, 3, 4, 5, 7, 8} д№ұеәҸйғЁеҲҶдёә {}

еӨҚжқӮеәҰеҲҶжһҗ:

ж—¶й—ҙеӨҚжқӮеәҰ:

дёҖе…ұжңүдёӨеұӮеҫӘзҺҜ, еҶ…еұӮеҫӘзҺҜ j зҡ„йҒҚеҺҶж•°йҮҸеҸ—еҲ° еӨ–еұӮеҫӘзҺҜ i зҡ„еҪұе“Қ

i = 0 ж—¶, j иҰҒд»Һ 0 йҒҚеҺҶеҲ° n-1, еҚі: n

i = 1 ж—¶, j иҰҒд»Һ 1 йҒҚеҺҶеҲ° n-1, еҚі: n-1

i = 2 ж—¶, j иҰҒд»Һ 2 йҒҚеҺҶеҲ° n-1, еҚі: n-2

вҖҰ вҖҰ

i = n-1 ж—¶, j иҰҒд»Һ n-1 йҒҚеҺҶеҲ° n-1, еҚі: 1

жүҖд»ҘдёҖе…ұиҰҒйҒҚеҺҶ: n + (n-1) + (n-2) + вҖҰ + 1 ж¬Ў

ж №жҚ®зӯүе·®ж•°еҲ—жұӮе’Ңе…¬ејҸ: (n+1)*n/2 = (n^2 + n)/2
