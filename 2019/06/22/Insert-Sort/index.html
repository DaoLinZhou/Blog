插入排序 希尔排序
代码复杂程度 简单 简单

插入排序 Insert Sort

思想: 把当前元素向前遍历, 如果前一个元素大于当前元素则swap

template <typename T>
void insertionSort(T arr[], int n){
    for( int i = 1; i < n; i ++ ){
        // 寻找元素 arr[i] 合适的插入位置
        for( int j = i; j > 0 && arr[j] < arr[j-1]; j -- )
            swap(arr[j], arr[j-1]);
    }
}

优化版插入排序

特点: 在数组中元素近乎有序的情况下, 时间复杂度进化为O(n)

思想: 保存当前元素, 向前遍历, 如果前一个元素大于大于当前元素, 则把当前元素赋值为前一个元素的值, 直到找到正确的位置, 把保存的值赋值给这个位置的元素

作用:

  1. 生成系统日志时, 日志本身就是按时间顺序, 但有可能会有一些乱序的元素
  2. 当数据量比较小时性能更好, 可以作为其他高级排序的子过程对其进行优化, 通常当数据集小于15, 就可以用插入排序进行优化

上一个版本的swap十分耗时, 因为一次swap 相当于三次赋值, 通过改进我们可以让它只交换一次

template <typename T>
void insertionSort(T arr[], int n){
    for( int i = 1; i < n; i ++ ){
        T temp = arr[i];
        int j;          // j保存元素temp应该插入的位置
        for( j = i; j > 0 && temp < arr[j-1]; j -- )
            arr[j] = arr[j-1];
        arr[j] = temp;
    }
}

希尔排序 (Shell Sort) – 插入排序的进化 O(n^(3/2))

插入排序: 每次和前一个元素进行比较

希尔排序: 每次和前h个元素进行比较 (h 从很大的值缩小到1)


思想: 将一个完全无序的数组 -> 变成近乎有序的数组 -> 变成有序性更强的数组 -> h = 1 变成排好序的数组

template <typename T>
void shellSort(T arr[], int n){
    // 计算increment sequence 1, 4, 13, 40, 121, 364, 1093...
    int h = 1;
    while( h < n/3 )
        h = 3 * h + 1;
    while( h >= 1 ){
        //h-sort to array
        for( int i = h; i < n; i++ ){
            // 对 arr[i], arr[i-h], arr[i-2*h], arr[i-3*h]... 使用插入排序
            T e = arr[i];
            int j;
            for( j = i; j >= h && e < arr[j-h]; j-= h)
                arr[j] = arr[j-h];
            arr[j] = e;
        }
        h/=3;
    }
}
