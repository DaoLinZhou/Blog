The Rule of Sum:

• If a first task can be performed in m ways

and a second task can be performed in n

ways, and the two tasks are mutually

exclusive (cannot be performed

simultaneously), then performing either

task can be accomplished in any one of m+n ways (Ex. 1-3)

The Rule of Product:

• If a task can be broken down into the first

and second stages, and if there are m

possible outcomes for the first stage and n

possible outcomes for the second stage,

then the whole task can be accomplished

in m·n ways (Ex. 1-4)

Maximal, minimal elements

Given a poset (A, R):

  • x ∈ A is a maximal element if (a != x) -> (x, a) !∈ R, ∀a∈A
    • 简单来说就是在Hasse 图中， 没有任何元素在这个元素之上
  • y ∈ A is a minimal element if (a != y) -> (a, y) !∈ R, ∀a∈A
    • 简单来说就是在Hasse 图中， 没有任何元素在这个元素之下

Greatest， least element

Given a poset (A, R):

  • x ∈ A is a Greatest element if (a, x) ∈ R, ∀a∈A

    • 即, 在Hasse图中， 所以元素(包括自己, 因为reflexable)都在这个元素下方

  • y ∈ A is a least element if (y, a) ∈ R, ∀a∈A

    • 即, 在Hasse图中， 所以元素(包括自己, 因为reflexable)都在这个元素上方

y is the greatest lower bound of B if ∀(x, y)∈R, where x is a lower bound of B

y is the least upper bound of B if ∀(y, x) ∈ R where x ia an upper bound of B

如果x是子集A的最小上界， x是一个上界，并且小于A的任何其他上界
