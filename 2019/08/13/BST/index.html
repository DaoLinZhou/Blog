树形结构, 但不一定是完全二叉树

性质: 左子树都比自身小, 右子树都比自身大

特点: 树形结构天然适合递归算法

插入/查找/ 删除等操作理想情况下都是O(log(n)) 的复杂度,

​ 特殊情况会退化为链表, O(n)级别, 也就是当顺序插入所有节点时

拓展

优化版数据结构: 2-3树 AVL树, 红黑树

平衡二叉树 和 堆 结合 : Treap

KD树, 区间树, 哈夫曼树

代码

注意, BST中 public 方法的返回值不应该为 Node, Node 是内部结构,不应该对外部开放(封装性).

理解 remove floor ceil successor predecessor 这四个方法的实现可以更深刻地理解BST的结构和递归的思想
