所需依赖

操作excel     pip install openpyxl
操作mysql        pip install mysqlclient
插入图片        pip install pillow

操作Excel

新建WorkBook和WorkSheet

一个WorkBook下可以有多个WorkSheet

wb = Workbook()

# 选中默认激活的表单
ws = wb.active
# 新建表单
ws_two = wb.create_sheet('我的表单')

# 设置ws的标题和颜色
ws.title = '你的表单'
ws.sheet_properties.tabColor = 'ff0000'

# 保存
wb.save('./static/test.xlsx')

插入数据(合并/设置字体)

ws['A1'] = 66
ws['A2'] = '你好'
ws['A3'] = datetime.now()

# 遍历
for row in ws_two['A1:E5']:
    for cell in row:
        cell.value = 2

# 对数据进行求和, 调用函数
ws['G1'] = '=SUM(A1:E4)'

# 插入图片 pip install pillow
img = Image('./static/temp.JPG')
ws.add_image(img, 'B1')

# 合并/取消合并 单元格
ws.merge_cells('A4:E5')
ws.unmerge_cells('A4:E5')

# 设置文字字号和颜色
font = Font(sz=18, color=colors.RED)
ws['A2'].font = font

读取数据

wb = load_workbook('./static/template.xlsx')
names = wb.sheetnames
# 拿到worksheet
# ws = wb.active
ws = wb[names[0]]

操作MySql

获取数据库连接

conn = MySQLdb.connect(
    db='test',
    host='localhost',
    user='root',
    password='123456',
    charset='utf8'
)

# 拿到游标
curcor = conn.cursor()
# 设置自动提交sql语句
conn.autocommit(True)

执行sql

sql = '...'     # 省略具体语句
cursor.execute(sql)     # 执行sql, 注意只有设置autocommit才会真正执行
rows = curcor.fetchall()    # 获取返回信息

小结

由于操作excel时会经常使用A1 B8 等标识, 所以建议使用str的format函数进行拼接字符和数字
