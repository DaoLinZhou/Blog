如何保持字典有序(py3.5)

python3.5及一下版本中 dict是无序的, 就是说放入元素的顺序和迭代的顺序不同, 因此如果想要保持字典有序, 应该实用collections.OrderedDict

from collections import OrderedDict
from itertools import islice

od = OrderedDict()
od['c'] = 1
od['b'] = 2
od['a'] = 3

print(od.keys())

和普通的字典一样, OrderedDict不支持切片操作

所以可以使用itertools下的islice 对这两种字典进行切片

od = OrderedDict()
od['c'] = 1
od['b'] = 2
od['a'] = 3

# 切片[1:2]
print(list(islice(od, 1, 2)))

d = {}
d['c'] = 1
d['b'] = 2
d['a'] = 3

# 切片[0:2]
print(list(islice(d, 2)))

python3.6以上dict默认就是有序的字典, 但这不意味着这个类就没有用处, 如果我们希望代码能够向下兼容, 那么依然要使用OrderedDict作为有序字典

如何实现可迭代对象和迭代器对象

我们可以使用for-in来迭代一个对象

from collections import Iterable, Iterator

l = [1, 2, 3, 4, 5]

    print(x)

如果一个对象想要被for-in迭代的话, 必须时Iterable的一个实例, 即:

isinstance(l, Iterable)     # True

或者说, list是Iterable的一个子类

issubclass(list, Iterable)        # True

for 循环拿到可迭代对象之后, 会调用内置函数iter, 把这个对象传进去

而它的返回结果就是一个迭代器对象

iter(可迭代对象) 会生成一个调用器对象

每次调用生成的是不同的迭代器对象

那么iter是怎么工作的呢?

# 而Iterable的抽象方法就是__iter__

print(next(it))

# for循环会处理这个异常, 此时循环结束

之前也说过调用iter时每次生成的是不同的迭代器对象

接下来我们要实现一个迭代器和可迭代对象

首先先来看Iterator和Iterable的源码

在 Lib/_collections_abc.py 下

class Iterable(metaclass=ABCMeta):

    __slots__ = ()

    @abstractmethod
    def __iter__(self):
        while False:
            yield None

    @classmethod
    def __subclasshook__(cls, C):
        if cls is Iterable:
            return _check_methods(C, "__iter__")
        return NotImplemented


class Iterator(Iterable):

    __slots__ = ()

    @abstractmethod
    def __next__(self):
        'Return the next item from the iterator. When exhausted, raise StopIteration'
        raise StopIteration

    def __iter__(self):
        return self

    @classmethod
    def __subclasshook__(cls, C):
        if cls is Iterator:
            return _check_methods(C, '__iter__', '__next__')
        return NotImplemented

我们的Iterator要实现next和iter方法, Iterable要实现iter方法

import requests
from collections import Iterable, Iterator


class WeatherIterator(Iterator):
    def __init__(self, cities):
        self.cities = cities
        self.index = 0

    def __next__(self):
        # 如果不能继续迭代, 抛出StopIteration异常
        if self.index == len(self.cities):
            raise StopIteration
        # 拿到下一个city的名称
        city = self.cities[self.index]
        # 指向下一个位置
        self.index += 1
        return self.get_weather(city)

    def get_weather(self, city):
        # 爬取天气信息
        url = 'http://wthrcdn.etouch.cn/weather_mini?city=' + city
        r = requests.get(url)
        data = r.json()['data']['forecast'][0]
        return city, data['high'], data['low']


class WeatherIterable(Iterable):
    def __init__(self, cities):
        self.cities = cities

    def __iter__(self):
        return WeatherIterator(self.cities)

进行迭代

if __name__ == '__main__':

使用生成器函数实现可迭代对象


期望效果:

if __name__ == '__main__':
    for n in PrimeNumbers(1, 30):
        print(n)

解决方案:

实现一个类, 将该类的iter内置方法实现成生成器函数, yield所有我们期望的值


实现:

class PrimeNumbers(Iterable):
    def __init__(self, a, b):
        self.a = a
        self.b = b

    def __iter__(self):
        for k in range(self.a, self.b+1):
            if self.is_prime(k):
                yield k

    def is_prime(self, k):
        # if k < 2:
        #     return False
        # for i in range(2, math.sqrt(k)):
        #     if k % i == 0:
        #         return False
        # return True
        return False if k < 2 else all(map(lambda i: k % i, range(2, int(math.sqrt(k))+1)))

all 会返回True如果参数中所有元素都为True, 这里我们抛弃了传统的写法, 而是使用更加有python风格的写法


如何 进行/实现 反向迭代

如何进行反向迭代?

  1. reverse之后再迭代 (缺点: 改变原本元素的位置)
  2. 通过切片[::-1]取得反向列表后再迭代 (缺点: 占用新的空间)
  3. 使用 reversed() 函数, 它将调用类内部的reversed方法返回一个反向迭代器, 使用它进行迭代

实现

from decimal import Decimal

class FloatRange:
    def __init__(self, a, b, step=0.1):
        self.a = Decimal(str(a))
        self.b = Decimal(str(b))
        self.step = Decimal(str(step))

    def __iter__(self):
        t = self.a
        while t <= self.b:
            yield float(t)
            t += self.step

    def __reversed__(self):
        t = self.b
        while t >= self.a:
            yield float(t)
            t -= self.step

yield是上面写过的使用生成器函数实现可迭代对象

这里我们要使用decimal.Decimal, 它和java中的BigDecimal很像, 当传入的是str类型的数字时, 可以进行精确的计算.

注意我们期望的是float值, 所以在yield时要转化成float

实现reversed内置函数就可以实现反向迭代

if __name__ == '__main__':
    fr = FloatRange(3.0, 4.0, 0.2)
    # 正向迭代
    for x in fr:
        print(x+1)

    print("-"*20)

    # 反向迭代
    for x in reversed(fr):
        print(x+1)

切片的原理 & 如何对迭代器进行切片


首先了解一下切片是如何工作的

# 生成一个list, 0-9
l = list(range(10))

# [] 运算符就是调用__getitem__
print(l[3])
print(l.__getitem__(3))

# 而切片操作依然使用的是[]运算符
# 不过传入的是slice对象
print(l[2:8])
print(l.__getitem__(slice(2, 8)))

所以可以看到, 如果进行迭代的对象没有getitem内置方法就不能进行切片

但是我们可以使用itertools.islice对可迭代对象进行切片

from itertools import islice

f = open('./word.txt', 'r', encoding='utf-8')

# 对f对象进行切片, 语义上是 f[5:9:4]
for line in islice(f, 5, 9, 4):
    print(line)

islice会从遍历0到指定结束位置的元素, 之后的元素不会遍历


仿写islice 帮助理解原理

def my_islice(iterable, start, end, step=1):
    tmp = 0
    for i, x in enumerate(iterable):
        if i >= end:
            break

        if i >= start:
            if tmp == 0:
                tmp = step
                yield x
            tmp -= 1


print(list(my_islice(range(100, 150), 10, 20, 3)))

如何在for语句里迭代多个可迭代对象

此时有两种情况, 并行迭代和串行迭代

并行迭代: 同时访问多个迭代器中的某一个位置的元素

串行迭代: 访问完一个迭代器中所有对象后可以接着访问下一个迭代器中的对象


并行迭代的实现

使用zip(), 传入多个可迭代对象, 它会以元组的方式进行配对

from random import randint

# 生成数据
chinese = [randint(60, 100) for _ in range(20)]
math = [randint(60, 100) for _ in range(20)]
english = [randint(60, 100) for _ in range(20)]

# zip的长度等于所有可迭代对象中最短的长度
zip(chinese, math, english)

t = []
for s1, s2, s3 in zip(chinese, math, english):
    t.append(s1 + s2 + s3)

# sum要传入可迭代对象, 我们就不用拆包了
t2 = [sum(s) for s in zip(chinese, math, english)]

t3 = list(map(sum, zip(chinese, math, english)))

对与比较简单的操作也可以使用map

# map 本身就支持并行操作
t4 = list(map(lambda s1, s2, s3: s1+s2+s3, chinese, math, english))

串行迭代的实现

使用itertools.chain , 传入多个可迭代对象, 它会把它们合并为一个可迭代对象

from itertools import chain

l = list(chain([1, 2, 3, 4], [5, 6, 7], [8, 9]))

print(l)
